# Autonomous UAV Landing System (A_UAV_L_S)

## 📝 Описание проекта
Данный проект представляет собой интеллектуальную систему для БПЛА (на базе COEX klever), предназначенную для автономного поиска, отслеживания и мягкой посадки на движущуюся цель. 

Основная особенность системы — использование **нейронных сетей** для прогнозирования траектории движения платформы, что позволяет компенсировать задержки связи и инерцию дрона.
